Recent Projects:

Coding Arduino robots to detect obstacles around them

In this project we used the Arduino C language to code the robot’s movement when it detected an object using its whisker sensors. We also worked on the hardware on this project, and attached three pins that detected when the whisker sensors pushed back. Once this detection was caught our robot was coded to go back for 3 seconds and them turn away from the object. We also added a red light feature, so as soon as the robot senses something in its way the user will be able to see the red light go off
